Este es un ejemplo de párrafo, aprovechando pongo una etiqueta que se llama "href" LINK PARA HACER MÁS PODEROSA TU PC si le picas a estas letritas azules te redirige a otra parte que tu desees, ya sea una URL u otra cosa que tu desees como otra página HTML.

La etiqueta "img" sirve para poner imagenes en tu html y esta va acompañada de otras etiquetas la cuales son "src" que sirve para que el código detecte el origen de la imagen ya sea un link o un archivo, el "alt" que sirve para poner un texto alternativo, y "width" y "heigth" que sirven para modificar el tamaño de la imagen.
